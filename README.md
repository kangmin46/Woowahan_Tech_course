* 웹 서버를 직접 구현해 HTTP를 이해하고, 서블릿 컨테이너의 동작 원리를 이해한다.
* 패킷을 직접 구현해 OSI 7 Layer 및 네트워크 통신 방식의 이해도를 높인다.
* JDBC 라이브러리, MVC, DI 프레임워크 구현을 통해 Spring 프레임워크의 동작 원리를 이해한다.
